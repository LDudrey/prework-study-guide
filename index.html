HTML

  • HTML is made up of elements that work together to create the structure, framework of a webpage. They ARE case-sensitive.
  • Elements that are nested inside other elements are called child elements. If there are three elements nested inside of a <body> element, those three nested elements are known as child elements.
  • The element that holds all the children elements is known as a parent element.
  • Types of elements:
    • Text elements
    • Image elements
    • Header elements
    • List elements
    • Meta-related elements
    • Content division elements
    • Semantic elements
  • Preceding HTML code is a declaration <!DOCTYPE html> Declarations are NOT case-sensitive. Declaration tells the browser what document type to expect when it is opened.
  • The HTML attribute <html lang="en"> tells the browser what language for the webpage.
  • HTML attributes are always defined in the beginning element, never the closing element.
  • The <head> element contains metadata information about the webpage.
  • The <meta> element defines metadata such as the character set, description, keywords, author, and viewport.
    • The meta element <meta charset="UTF-8"> defines the character set as UTF-8 which allows us to use most characters and symbols on the page.
    • The meta element <http-equiv> element sets an HTTP header for content.
    • The meta element <viewport> element sets the default view of our webpage.
    • The <title> element defines the title for a webpage. It is required and extremely important for Search Engine Optimization (SEO).
  • The <body> represents the visible content shown to the user. It is an example of a Semantic HTML element.
  • Use <h1> headings in order from <h1> to <h6>
  • The <p> represents a paragraph or block of text.
  • The <ul>, <ol>, <li> elements represent unordered lists, ordered lists, and list items.
  • The <img> element contains information about images that are displayed on the webpage. <src> defines the image location
  • The <br> element creates a line break or empty space.
  • The <a> stands for anchor and creates links to the same webpage or other pages.
  • The <style> element defines the CSS styles associated with a website.
  • The <base> element defines the base URL for a webpage.
  • The <link> element connects an external resource to the HTML document.
  • The <script> element embeds client-side scripts in the HTML document.

CSS

  • Inline CSS apply styles directly in the HTML file to specific HTML tag using a style attribute with a CSS rule to style a particular page element only.
    • Example:<h1 style="color:red;">Prework Study Guide</h1> This changes a single heading to red.
  • Internal Style Sheet contains CSS rules for the webpage in the head section of the HTML file. Rules only apply to that page and you can configure CSS classes and IDs.
  • Most common is External CSS style sheet. This is prefered because it allows developer to keep all their CSS rules in a seperate file to make changes easier.
  • The <link> element creates the connection to the exernal file.
  • To link an external CSS sheet add <link rel="stylesheet" href="./assets/style.css"> element after the final meta element and before title element.
    • The rel attribute specifies the relation ship between the current ocument and the linked document resource.
    • The href attribute specifies the location URL of the external resource. This href attribute is using a relative path. This means the location is dervied from the current location (index.html). We must lookin the assets folder to locate.
  • CSS box model refers to each HTML element is essentially a box. Each box consists of margins outside, padding and content.
    • A margin indicates how much space we want around the outside of an element.
    • A padding indicates how much space we want around the content inside an element.
  • To help us write Do Not Repeat Yourself (DRY) code. We can combine two selectors into a single CSS rule with a comma allowing the declarations to affect both selectors.
  • The text-align is a way to center a title or heading on a webpage
  • The <img> selector assigns property values to any image element in the markup.
  • The .card selector contains width, margin, border and box-shadow properties.
  • The period preceding card designates this as a class selector. Any HTML element assigned to the class with gain these property assignments.
  • Assigning colors in CSS can be done by semantic references such as gray or blue.
  • Color can also be added by its hexadecimal code or hex #.

Git

  • Git is a version control system that lets you manage and keep track of each version of your project.
  • GitHub is a cloud-based hosting service that allows us to keep track of our projects inside repositories that are stored remotely.
  • cd: change directory, navigates through the directory
  • cd ..: navigates backwards through the directory one level at a time
  • ls: displays a list of files and folders in current directory
  • pwd: displays the current working directory
  • mkdir: make directory, creates a new directory
  • start: opens a file Ex: start first-day.txt
  • git status: checks what branch we are currently on
  • git checkout main: takes you to main branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: will add all changes to next push
  • git commit -m "your message here": adds a message to describe the changes you made during coding session.
  • git pull origin main: syncs your local and remote branches
  • git push origin branch-name: sends your changes to github branch
  • touch file.name: will create a new file in the current folder
  • start file.name: opens file

JavaScript

  • You link the .js file under the <footer> element within the <body> on the html page.
    • Ex: <script src="./assets/script.js"></script>
  • Control flow is the order in which a computer executes code in a script. JavaScript is read from top to bottom.
  • Each line of code ends with a semicolon.
  • Strings are contained in quotation marks.
  • A variable (var) is a named container that allows us to store data in our code. After the var keyword we give the variable a name.
    • Ex: var shape
  • = is an assignment operator to assign specific data we store in our variable.
    • Ex: var shape = "circle";
  • An Array is a single variable that is used to hold a group of data. Data is typically related in some way.
    • Ex: var shapes = ["triangle", "square", "pentagon", "circle"];
  • Another Array example: var studentInfo = ["Lu", 54, true];
    • Only student name is in quotations because it is a string
    • 54 is a number and does not need quotations
    • true is a Boolean value, no quotations needed.
  • if and else if statements are a way of creating a conditional statement in JavaScript.
  • A value is considered falsy if it is one of the following values: 0, -0, On, "", null, undefined, NaN or a Boolean data type with false value.
  • A value is considered truthy if all other values including Boolean are true.
  • for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
